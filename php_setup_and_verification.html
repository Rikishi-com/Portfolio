PHPの動作確認と基本的な設定

このページでは，Apacheサーバ上でPHPを動作させるための確認手順と，MariaDB，phpMyAdminの設定をまとめます．

この手順は /var/www/html をドキュメントルートとし，ApacheとPHPが正常にインストール・起動されていることを前提とします．

1. PHPの動作確認

phpinfo()で環境確認

  • /var/www/html 配下に phpinfo.php というファイルを作成する．
sudo nano /var/www/html/phpinfo.php

nano-テキストエディタ．vim,emacsより操作が直感的


例：ファイルを開く（なければ作成）

nano filename.txt

以下の内容を記述して保存する． 
<?php
phpinfo();
?>

ブラウザで以下にアクセスする： 
http://[EC2のパブリックIP]/phpinfo.php
  • PHPの詳細情報ページが表示されれば，PHPは正しくインストールされている．

Hello World をPHPで表示

  • /var/www/html/hello.php というファイルを作成する：
sudo nano /var/www/html/hello.php

以下のコードを記述： 
<?php
echo "Hello World";
?>

echo-文字列や変数の値を画面に表示するコマンド


例：コンソールにHello Worldと表示される

echo "Hello World"

ブラウザで確認： 
http://[EC2のパブリックIP]/hello.php
  • Hello World と表示されれば成功です．

2. MariaDBのインストール

パッケージのインストール 

sudo dnf install -y mariadb105-server

mariadb105-server-MariaDB10.5系のサーバパッケージ

サービスの起動と自動起動設定 

sudo systemctl start mariadb
sudo systemctl enable mariadb

初期設定スクリプトの実行 

sudo mysql_secure_installation
  • パスワード設定，匿名ユーザー削除，リモートログイン制限，テストDB削除などを設定できます．
  • 対話形式で安全に初期設定を行えるコマンド

3. phpMyAdminのインストール

EPELリポジトリの有効化とインストール 

sudo dnf install -y epel-release
sudo dnf install -y phpMyAdmin

Apache設定ファイルの編集（アクセス許可） 

sudo nano /etc/httpd/conf.d/phpMyAdmin.conf
  • Require ip 127.0.0.1 や Allow from 127.0.0.1 を Require all granted に変更することで外部からのアクセスを許可できる

Apacheの再起動 

sudo systemctl restart httpd

4. phpMyAdminへログイン

  • ブラウザで以下にアクセス：
http://[EC2のパブリックIP]/phpMyAdmin

  • MariaDBのユーザー名とパスワードを入力してログイン．

  • ログインできれば，Apache・PHP・MariaDB・phpMyAdmin の連携確認は完了．
